# devlog/notes 子文件夹说明

> 本目录存放按需读取的详细备忘与会话归档。主索引在 `../session-notes.md`。

## 读取协议

1. **新会话启动**：只读 `../session-notes.md`（主索引）+ `../blockers.md`，**不要默认读取本目录文件**
2. **按需读取**：根据主索引的"会话历史索引表"或"核心机制"链接，定位到具体文件后再读取
3. **搜索优先**：如需查找特定主题，用 `rg` 搜索本目录关键词，定位到具体文件与行号后再读

## 新增会话归档规则

每次会话结束后：
1. 当天已有归档文件 → 追加到 `YYYY-MM-DD.md` 末尾
2. 当天无归档文件 → 新建 `YYYY-MM-DD.md`
3. 在主索引 `../session-notes.md` 的"会话历史索引表"追加一行（日期 | 主题 | 摘要 | 文件链接）
4. 更新主索引的"最近会话重点"为本次会话
